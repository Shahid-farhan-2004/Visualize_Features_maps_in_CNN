# CNN Feature Map Visualization using PyTorch

## Overview

This project demonstrates how to visualize **feature maps** generated by the first convolutional layer of a Convolutional Neural Network (CNN). The model processes images from the **CIFAR-10** dataset, and the activations from the first convolutional layer are displayed using Matplotlib.

---

## Features

- Loads the CIFAR-10 test dataset
- Builds a simple CNN model
- Extracts feature maps from the first convolutional layer
- Visualizes the first six feature maps
- Uses Matplotlib for visualization

```python
transform = transforms.Compose([

    )

```

---

## Feature Map Extraction

The feature maps are obtained from the first convolutional layer.

```python
with torch.no_grad():
    feature_maps = F.relu(model.conv1(image))
```

The output shape is:

```
(batch_size, 16, 32, 32)
```

where:

- `batch_size` = Number of images in the batch
- `16` = Number of filters in the first convolution layer
- `32 × 32` = Size of each feature map

---

## Feature Map Visualization

The first six feature maps of the first image are displayed.

```python
fig, axes = plt.subplots(1, 6, figsize=(15, 3))

for i in range(6):
    axes[i].imshow(feature_maps[0, i].cpu(), cmap="viridis")
    axes[i].axis("off")

plt.suptitle("Feature Maps from Conv1")
plt.show()
```

## How It Works

1. Load the CIFAR-10 dataset.
2. Create a CNN model.
3. Pass a batch of images through the first convolution layer.
4. Apply the ReLU activation function.
5. Extract the generated feature maps.
6. Display the first six feature maps for the first image.
